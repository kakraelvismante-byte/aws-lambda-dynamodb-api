# AWS Lambda + DynamoDB API Project

## 🚀 Project Overview

This project is a serverless backend API built using AWS Lambda and DynamoDB. It allows users to send messages via a URL and stores them in a NoSQL database.

## 🛠️ Technologies Used

* AWS Lambda
* Amazon DynamoDB
* Python (boto3)

## ⚙️ Features

* Save messages via API endpoint
* Automatically generates unique IDs
* Stores data in DynamoDB

## 📌 Example Usage

https://your-api-url/?message=hello
